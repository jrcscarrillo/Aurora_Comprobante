Seleccion de FacturasTipo: Proceso Mixto

**** OJO **** Antes de realizar este proceso se debe haber procesado la sincronizacion de la base de datos

Primer paso seleccionar las facturas (por fechas, por numero, otro) por definir

Nada adicional

Objetivos

Seleccionar todas las facturas que seran procesadas con el certificado digital. Resultado: Numero de facturas seleccionadas, total facturado.

Selecciona Facturas

Firmar FacturasTipo: Proceso Internet

**** OJO **** En este proceso se debe tener una linea de internet con buena velocidad y rendimiento

Primer paso insertar las claves y el certificado digital en el lote de facturas (puede ser una factura)

Segundo paso: ingresar el nombre del archivo a generar

Tercer paso: generar el archivo xml y transmitirlo al SRI el archivo se guarda en el directorio LETRA DEL DISCO DEL SERVIDOR:/wamp/www/aurora/tmp/NOMBRE DEL ARCHIVO

Cuarto paso se recibe la respuesta del SRI

Quinto paso se actualizan las facturas autorizadas o no autorizadas

*** Sexto paso: enviar las facturas autorizadas al correo electronico de cada cliente

Nada adicional

Objetivos

Utilizar los servicios del internet provistos por el SRI para autorizar las facturas con la firma digital
